> **Optimizing today's logistics to protect tomorrow's environment.**

## Project Context
This repository contains the research and algorithmic development work carried out by **CesiCDP** as part of the call for expressions of interest launched by **ADEME** (French Agency for Ecological Transition). 

Facing current challenges in transport logistics (distribution, delivery, waste collection), the goal of this project is to design a demonstrator for intelligent multimodal mobility.

## Technical Objective
The core of the problem relies on solving a variation of the **Traveling Salesman Problem (TSP)** applied to road networks. 

The algorithm developed here aims to:
1. Calculate an optimal route connecting a specific subset of cities.
2. Return to the starting point.
3. **Minimize the total duration of the tour** (and consequently, lower fuel consumption and polluting emissions).

## Tech Stack & Methodology
* **Language:** Python 3 (PEP 8 compliance)
* **Format:** Jupyter Notebooks (focusing on an explanatory approach and storytelling)
* **Methodology:** Mathematical modeling, time-space complexity analysis, and statistical study of experimental performances.
